# Teste_Pi

Estrutura de trabalho:

- `main`: integração final
- `back-end`: backend
- `front-end`: frontend
- `docs`: documentação

Regras:

1. Cada branch trabalha só na sua pasta.
2. Não trocar de branch dentro da mesma pasta.
3. Fazer commits pequenos e frequentes.
4. Abrir PR para `main` quando o bloco estiver pronto.
